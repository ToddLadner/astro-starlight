Easily create nice looking color-adaptive, responsive, and accessible buttons.

## 9 Button types

Unified modern style, visual differences reinforce purpose. Keyboard, mouse & touch ready.

## Usage

To apply this component, use the `<Button />` component and, if needed change the type and add a modifer to adjust the theme, size, and shape of the button.

**Ready to customize**

| Type   | Description                        |
| ------ | ---------------------------------- |
| button | The default value renders a button |
| submit | Renders a submit button            |
| reset  | Renders a reset button             |
| file   | Renders a submit button            |
