Easily create nice looking color-adaptive, responsive, and accessible buttons.

## Usage

To apply this component, use the <code>&lt;Button&gt;</code> component add a modifer to adjust the theme size and shape of the button.

<Tabs>
  <TabItem label="PREVIEW">
    <div class="flex items-center not-content gap-sm">
      <Button value="Submit" primary />
      <Button value="Button" secondary sm pill />
      <Button value="Button text" secondary lg disabled />
    </div>
  </TabItem>
  <TabItem label="MARKUP">
    ```js
    <Button value="Submit" primary />
    <Button value="Button" secondary sm pill />
    <Button value="Button text" secondary lg disabled />
    ```
  </TabItem>
</Tabs>
